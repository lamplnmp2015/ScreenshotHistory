📸 Screenshot History

常驻系统托盘的截图历史管理工具。自动记录你的每一张截图，并能通过截图里的文字把它找回来。

核心功能

  • 自动捕获 — 微信/QQ/PrintScreen/Snipping Tool 等工具截图后自动保存，复制图片不会入库
  • OCR 全文搜索 — 中英文识别（内置 Tesseract），按截图里的文字搜索历史
  • 时间线浏览 — 倒序缩略图网格，高分屏下依然清晰
  • 详情预览 — 滚轮缩放 · 拖拽平移 · 双击复位 · 显示文件名/来源应用
  • 灵活删除 — 可只删历史记录，或连磁盘图片一起删
  • 系统托盘 + Ctrl+Shift+H 全局热键

开箱即用

安装包已内置 Tesseract OCR 引擎和中英文语言模型，安装后无需任何额外配置，OCR 立即可用。

技术栈

Tauri 2 Rust Vue 3 TypeScript Pinia SQLite FTS5 Tesseract OCR Windows
