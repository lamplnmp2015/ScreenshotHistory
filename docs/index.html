功能特性

🤖
自动捕获
后台监听剪贴板，微信/QQ/PrtScn 截图后自动保存，无需任何手动操作。
🔍
OCR 全文检索
中英文混合识别，按截图里的文字即可搜索。Tesseract 内置，开箱即用。
🎯
智能去噪
自动区分截图与复制的图片，只记录真正的截图。置信度过滤剔除图标乱码。
🔎
滚轮缩放预览
详情页支持滚轮缩放、拖拽平移、双击复位，随时放大查看截图细节。
🗂
按日期归档
截图按日期自动整理到本地目录，随时可在资源管理器中找到原图。
⌨️
全局热键
Ctrl+Shift+H 随时显隐窗口，常驻系统托盘，不打断任何工作流。
